# Contributing to iZeroPDF

Thank you for considering contributing to iZeroPDF! We welcome contributions from everyone.

## How to Contribute

### 1. Report Bugs
- Use the [GitHub Issues](https://github.com/AmiNilay/iZeroPDF/issues) page
- Check if the bug has already been reported
- Include clear steps to reproduce the bug
- Include browser name and version

### 2. Suggest Features
- Open a new issue with the label "enhancement"
- Describe the feature and why it would be useful
- Include examples if possible

### 3. Submit Pull Requests
1. Fork the repository
2. Create a new branch: `git checkout -b feature/your-feature-name`
3. Make your changes
4. Test your changes in multiple browsers
5. Commit with a clear message
6. Push to your fork
7. Open a Pull Request

## Code Style
- Use vanilla JavaScript (no frameworks)
- Write clear comments
- Use camelCase for variables and functions
- Use meaningful variable names
- Keep functions focused on a single task

## Testing
- Test in Chrome, Firefox, and Safari
- Test on mobile devices if applicable
- Ensure all features work offline
